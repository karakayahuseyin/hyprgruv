## Archlinux + Hyprland Gruvbox Customization

![](view.png)

### Default Apps
- terminal : kitty
- file manager : Thunar
- menu : wofi
- bar : waybar
- browser : chromium
- wallpaper : hyprpaper

### Essentials
- cava : sound visualizer
- zathura : pdf reader
- nnn : terminal file explorer
- obsidian : note taking
- neovim : code editor
- hyprshot : screenshot
- neofetch : fetch
- swappy : image viewer
- btop++ : system monitoring
